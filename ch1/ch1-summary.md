# Chapter 1: A Tutorial Introduction

## 1.1 Getting Started

* `stdio.h` is required to use `printf`.
* `main()` should be `int main()`.
* `cc` is the compile command.

To compile a C source code:
```bash
cc hello.c
```

To execute the compiled file `a.out` in the working directory:
```bash
./a.out
```
